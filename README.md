## Tentang Repositori Ini

Ini adalah pertama dan terakhir kali saya mengerjakan projek Uji Kompetensi Keahlian di sekolah Wikrama 1 Garut, tema aplikasi ini adalah SPP Siswa yang memiliki fungsi sebagai berikut:

- Siswa
Dapat melihat riwayat pembayaran dan status SPP perbulan serta total tagihan dari SPP tersebut.

- Petugas
Dapat membuat dan mengelola tagihan dan pembayaran dari siswa terkait.

- Admin
Seperti layaknya admin, dapat mengelola semua data siswa, petugas, Jenis SPP, Harga, SPP perbulan, dan tagihan.

## Instalasi

1. Clone repositori ini ke lokal Anda.
2. Install dependensi Laravel.
- `composer install`
- `npm install`
3. Konfigurasi database dan env.
4. Migrasi database.
- `php artisan migrate`
5. Seed database.
- `php artisan db:seed`
6. Jalankan aplikasi.
- `php artisan serve`
- `npm run dev`
